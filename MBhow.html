Main screen

The main screen contains three main features. First is practice. It will show a game of easy 4x4 difficulty type of puzzle objects. On transition screen you will be informed when game starts. The results of this will not be saved since your not logged in. It is just there for you to sharpen your skills. Second feature is unavailable on casual and you need to be logged in to play competative. Finally the third feature is a quick read to know what memory battle is all about and how it is played. Left corrner grands you the ability to sign up if your already signed up you can use Log in in the right upper corrner.

Sign up & Log in

You will have to pick your username, password, email and a secret answer which has to be remembered at all times in case you forget your password. Click on submit it will inform you you have created new profile. Sing up screen will inform you if username is unavailable also if your input is invalid or missing. Button is available to transition you dirrectly to a log in screen. In Log in screen use the created password and username and click submit it will inform you that you are logged in and swich you to a main screen. You have a dirrect button if you forgot your password.

Competative mode

Once a person is logged in it will unlock the competative option. Person will be able to log out at the bottom of a screen and it will be remided if he is infact logged in at the top of the screen. On this mode practice option is disabled which you can access again after logging out.

How to play on competative mode ?

After user clicked on competative button he will enter a selection screen. There user will be able to construct the game. There are two difficulty options easy and hard mode. Easy mode is 4x4 puzzle that contains 8 pairs user must uncover. Hard mode is a 5x6 puzzle that contains 15 pairs. User can also select number of players that will play the game. He can chose between 2, 3, 4 players. Pressing ready will start the game. Players will have 2 seconds or 3 seconds reveil of puzzle pictures depending on difficulty type. User can resign a game if he does not want his game results saved which will return him back to the main screen. Score is kept at the bottom of puzzles. Players gets 1 point if they guess correctly and -0.5 points for a wrong guess. Player doesn t need to uncover all pairs if he is does not want to. Once he is satisfied with his score he presses finish. After 3 seconds next player will start the game. After all players have compleated thier games finish button will save the game and its results and display them on the main screen popup window.
